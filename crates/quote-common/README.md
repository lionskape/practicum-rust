# quote-common

Общие типы и константы протокола, разделяемые между `quote-server` и `quote-client`.

## Содержимое

### Структуры данных

- **`StockQuote`** — одна котировка акции (тикер, цена, объём, временная метка), передаваемая по UDP в формате JSON.

### Ошибки

- **`ProtocolError`** — ошибки уровня протокола: невалидная команда, неизвестный тикер, ошибка адреса, IO.

### Константы протокола

| Константа              | Значение  | Описание                                  |
|------------------------|-----------|-------------------------------------------|
| `CMD_STREAM`           | `STREAM`  | Префикс команды подписки                  |
| `RESP_OK`              | `OK`      | Префикс успешного ответа                  |
| `RESP_ERR`             | `ERR`     | Префикс ответа с ошибкой                  |
| `PING_PAYLOAD`         | `b"PING"` | Содержимое keepalive-пакета               |
| `PING_INTERVAL_SECS`   | `2`       | Клиент отправляет PING каждые N секунд    |
| `PING_TIMEOUT_SECS`    | `5`       | Сервер отключает клиента без PING через N с|
| `GENERATION_INTERVAL_MS`| `100`    | Генерация котировок каждые N мс (10 Гц)   |
| `UDP_BUF_SIZE`         | `4096`    | Максимальный размер UDP-датаграммы        |
